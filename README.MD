# Notas:

Uso de comandos en la consola con Node y Yargs

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -n, --num      Es el numero que se desea multiplicar       [number] [required]
  -l, --listar   Muestra la tabla de multiplicar en consola
                                                      [boolean] [default: false]
  -h, --hasta    Este es el numero hasta donde quieres multiplicar
                                                          [number] [default: 10]
```
